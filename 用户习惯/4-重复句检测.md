# 重复句检测

## 定义

相邻句子（被静音分隔）开头相同 ≥5字，通常是说错后重说。

## 核心原则

**先分句，再比对**：必须先按静音切分成句子列表，然后比对相邻句子。

## 两种重复类型

### 1. 开头重复（最常见）

相邻句子开头相同：
```
"这是我的第一个视频" → "这是我的第一个视频"
"我今天要讲三个内容" → "我今天要讲三个内容"
```

### 2. 中间重复（容易漏检！）

重复内容在句子中间，不只是开头：
```
"现在金融圈可火了，想扩大自己事业的这可是个大" → 
"想扩大自己事业的这可是个大好机会呢"
```
**这两句都有"想扩大自己事业的这可是个大"，是重复！**

## 检测方法

### 步骤

1. **按静音切分句子**（静音≥0.5s作为分隔）
2. **比对相邻句子**：
   - 检查开头是否相同（≥5字）
   - **也要检查中间是否有较长相同内容（≥10字）**
3. **比对隔一句**（中间是残句时，也要检查前后）

## 检测逻辑

```javascript
// 检查开头重复
function checkStartRepeat(curr, next) {
  if (curr.text.slice(0, 5) === next.text.slice(0, 5)) {
    return true; // 开头重复
  }
  return false;
}

// 检查中间重复（新增！）
function checkMiddleRepeat(curr, next) {
  const minLength = 10; // 至少10个相同字符
  
  // 检查 curr 的任意子串是否出现在 next 中
  for (let len = minLength; len <= curr.text.length; len++) {
    for (let start = 0; start <= curr.text.length - len; start++) {
      const substr = curr.text.slice(start, start + len);
      if (next.text.includes(substr)) {
        return true; // 中间发现重复
      }
    }
  }
  return false;
}

// 完整检测
if (checkStartRepeat(curr, next) || checkMiddleRepeat(curr, next)) {
  // 排除列表序号
  if (!isListNumbering(curr.text) && !isListNumbering(next.text)) {
    markAsError(shorter); // 删除整句
  }
}
```

## 排除场景：列表序号

**以下情况不是重复，不要删除**：

| 序号类型 | 示例 |
|----------|------|
| 中文数字 | 一、二、三、 |
| 阿拉伯数字 | 1.、2.、3.、 |
| 第一/第二 | 第一点、第二点 |

## 案例

### 开头重复（应删除）

| 前句 | 后句 | 删除 |
|------|------|------|
| "这是我剪出来的一个案例" | "这是我剪出来的一个案例" | 前句 |
| "我们就可以看到这里新的视频" | "我们就可以看到这里新的视频了" | 前句 |

### 中间重复（应删除，上次漏检！）

| 前句 | 后句 | 删除 |
|------|------|------|
| "...想扩大自己事业的这可是个大" | "想扩大自己事业的这可是个大好机会呢" | 前句 |
| "今天天气很好，我想去跑步" | "我想去跑步，但是下雨了" | 前句（有"我想去跑步"） |

### 列表序号（不应删除）

| 前句 | 后句 | 处理 |
|------|------|------|
| "一、有自有资金投资..." | "二、有自有资金投资..." | 不删除 |

## 重要提醒

```
❌ 只检查开头重复
✓ 也要检查中间重复（≥10字相同）

检查中间重复的方法：
1. 取前句任意10+字子串
2. 看后句是否包含这个子串
3. 包含就是重复
```
